# Fraud Detection in Financial Transactions

## Overview

This project is a Machine Learning-based Fraud Detection System that identifies fraudulent financial transactions using Logistic Regression.

The project uses the PaySim dataset and provides an interactive Streamlit web application where users can upload datasets and perform binary classification predictions.

---

## Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Logistic Regression Model
- Fraud Detection Prediction
- Confusion Matrix Visualization
- ROC Curve Analysis
- Accuracy, Precision, Recall and F1 Score
- Interactive Streamlit Dashboard
- User Dataset Upload Support

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn
- Streamlit

---

## Dataset

The project uses the PaySim Dataset, a synthetic financial transaction dataset designed for fraud detection research.

Features include:

- Transaction Type
- Transaction Amount
- Sender Balance
- Receiver Balance
- Fraud Label

---

## Machine Learning Workflow

1. Data Collection
2. Data Cleaning
3. Feature Engineering
4. One-Hot Encoding
5. Train-Test Split
6. Logistic Regression Training
7. Model Evaluation
8. Deployment using Streamlit
